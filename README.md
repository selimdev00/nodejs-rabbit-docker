## Deployment

To deploy this project run

```bash
  npm run start
```


## Run Locally

Clone the project

```bash
git clone https://github.com/selimdev00/nodejs-rabbit-docker
```

Go to the project directory

```bash
  cd nodejs-rabbit-docker
```

Start microservices on docker

```bash
  npm run start
```
